== Description ==
Let you import posts from your own Facebook group or from a public Facebook group using the Facebook API.

== Installation ==
Here's how to use this plugin:

1. Log in to your WordPress site administrator panel.
2. Go to 'Plugins' and press the 'Add New' button.
3. Click "Choose file" ("Browse") and select the downloaded zip file of the plugin.
4. Click the "Install Now" button.
5. Click on "Activate Plugin" button.
6. Go to Facebook Developers (here: https://developers.facebook.com/) and create New App.
7. Go to 'FB Group Import' and insert your App ID and App Secret.
8. On the same page you'll see 'Access Token' field with a link below it. visit this link and copy your App Token.
9. That's it! Now you can go to the Import page and import your Facebook group posts to your WordPress site.
